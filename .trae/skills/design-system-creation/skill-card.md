## Description: <br>
Complete workflow for creating distinctive design systems from scratch, including aesthetic documentation, token architecture, components, motion, and loading states. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[wpank](https://clawhub.ai/user/wpank) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and designers use this skill to start or refactor a product design system with documented visual direction, reusable design tokens, component primitives, motion patterns, and loading states. <br>

### Deployment Geography for Use: <br>
Global <br>

Mitigation: Install only from trusted sources and verify package or source provenance before use. <br>
